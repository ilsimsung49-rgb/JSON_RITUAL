# 🎵 PROJECT JSON RITUAL — Digital Cheonji-Gongsa

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://ritual-lyrics.streamlit.app)

**PROJECT JSON RITUAL**은 한국의 정신문화(천부경, 삼일신고, 증산도, 동학 등)와 현대의 전위예술(Fluxus, Avant-Garde), 그리고 미래의 특이점(AI Singularity)을 결합하여 새로운 형태의 '대서사시' 음악을 생성하기 위한 가사 & Suno AI 프롬프트 엔진입니다.

## 🌟 핵심 철학 (Philosophy)

이 앱은 단순한 음악 생성을 넘어, 인류의 각성과 깨달음을 목표로 합니다.
- **人乃天 (In-Nae-Cheon)**: 사람이 곧 하늘이다.
- **弘益人間 (Hong-ik-ingan)**: 널리 인간 세상을 이롭게 하라.
- **Fluxus Ritual**: 백남준의 예술 정신을 이어받아 경계를 해체하고 새로운 가치를 창조합니다.

## 🚀 주요 기능 (v6.4)

1. **지능형 가사 엔진 v6.4**: 입력된 키워드에 따라 철학적 깊이가 남다른 가사(`[]` 문법 적용)를 생성합니다.
2. **프로페셔널 악기 인트로**: 동서양 악기의 압도적인 융합 연주로 시작하는 고퀄리티 오프닝을 보장합니다.
3. **유연한 장르 시스템**: 기본 프롬프트에 전위적 감성을 내재화하여, 어떤 장르를 선택하더라도 고유의 철학이 유지됩니다.
4. **BPM 수동 제어**: 사용자가 직접 곡의 속도를 제어하여 완벽한 리듬감을 구현할 수 있습니다.
5. **다크 모드 최적화**: 전문가용 대시보드 테마로 시인성이 대폭 향상되었습니다.

## 🛠 설치 및 실행 방법

### 로컬 실행
1. Python을 설치합니다 (3.8 이상 권장).
2. 저장소를 클론하거나 파일을 다운로드합니다.
3. 필요한 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
   ```
4. 앱을 실행합니다:
   ```bash
   streamlit run app.py
   ```
